# savingBiocObjects

Documentation and guidance on strategies for saving and sharing Bioconductor
objects.

Topics covered:

- R serialization with `saveRDS()` and cross-release stability
- HDF5-backed storage for large assay data
- The _ArtifactDB_ ecosystem for cross-language serialization
- Use of BED format for genomic ranges
- Preserving object metadata alongside saved files
